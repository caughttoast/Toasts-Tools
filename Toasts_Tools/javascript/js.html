What is Javascript?

Javascript is what makes the webpages you visit function. For our case, it is what helps us exploit the policies created by districts. Bookmarklets are bookmarks with embeded Javascript code. This code can be anything from telling you the time to changing Google Classroom into an incognito search engine. However, it can only be used if developer tools are enabled. To test for this, you can press F12 or Ctrl+Shift+I. If a menu appears, it's enabled, if not, you are unable to use anything on this page unless you create a seperate HTML website.

To use any of the below as a bookmarklet, you need to place them in your bookmarks bar with the prefix javascript:. If using any of these in a website, remove the prefix.
